# 关于机器的升级槽

### 注意到mek、热力之类的mod提供了机器升级选项，因此添加该系统

资源值分为CPU与能量栅格
升级槽分为高能量槽，中能量槽，低能量槽，改装件槽，子系统槽（详见下）

一个单方块机器、多方块机器的主方块有资源值、升级槽的设定

一个“机器升级”有三个属性：CPU、能量栅格、槽位类型

因此：
低资源的机器无法安装过强的“机器升级”
但一些机器尝试可以通过增加资源的“机器升级”来支撑一个过强的“机器升级”

## 值得注意的是，该系统与舰船配置系统类似

因此，可以保持风格和逻辑上的统一

该系统应当在工业第二阶段时逐渐加入

## 关于子系统

我们注意到，在格雷的多方块中，有一类特别的多方块

如丹尼尔蒸馏厂（是叫这个名字吧？）

此蒸馏厂是由普通的蒸馏厂主方块中放入丹尼尔蒸馏厂升级芯片得来的

也就是类似于“机器进阶”的感觉

子系统升级也就是对应实现这种功能

## 此处是一个机器的例子

假设一个卷板机的槽位为 2改装件 2高能量槽 0中能量槽 2低能量槽 100CPU 200能量栅格

存在改装件：

CPU升级 -50能量栅格 +100CPU

能量栅格升级 -75CPU +100能量栅格

高能量槽：

加工速度升级 -150CPU +25% 加工速度

节能升级 -100能量栅格 -15% 能量消耗

低能量槽：

静音升级 -10 CPU -10能量栅格

能量栅格管理 -50 CPU +10%能量栅格输出

### 那么，在需要追求速度的情况下，我们希望高槽装配两个“加工速度升级”
于是此时消耗300CPU，这个机器-200CPU的情况下无法正常工作

应当安装两个“CPU升级”，此时获得了总体0CPU100能量栅格的速度最大化配装

### 如果我们希望高槽装配两个“节能升级”和一个“静音升级”呢？易得此时可以再安装“能量栅格管理”来解决问题
### 这仅仅是对于配装的简陋模拟，如果系统完善，将会有更多有趣的提升方法

# 一些备注
改装件安装后无法拆除

对于这套系统，还有一点：它可以替代掉覆盖板

也就是说，不再是将“传送带”覆盖板贴在机器上，而是将“传送带升级”插入该机器的高槽并配置工作和面，这使得覆盖板不再受限于六面的限制，支持更复杂的自动化操作，代价是受到机器资源的限制

对于输入和输出仓也应当有一个资源的系统来支持覆盖板。

## 该系统在舰船、星舰、太空建筑的系统设计上已经完善，且考虑到我们将以多方块形式来表现他们，这套系统在多方块结构主方块上的部署是必须的，因此讨论仅限于对工业机器的应用是否合适。
